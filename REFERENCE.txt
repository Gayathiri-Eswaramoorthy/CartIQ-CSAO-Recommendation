SYSTEM ARCHITECTURE
═══════════════════════════════════════════════════════════════════════════════

DATA PIPELINE                 TRAINING PIPELINE
─────────────────            ──────────────────

Raw CSVs                      Init Model
  ↓                             ↓
Parse Timestamps              Forward Pass
  ↓                             ↓
Extract Features              Compute Loss (BCE)
  ↓                             ↓
Merge Features                Backprop
  ↓                             ↓
Temporal Split                Update Weights
  ↓                             ↓
Normalize                      Validate
  ↓                             ↓
Pad Sequences                 Check Early Stop
  ↓                             ↓
PyTorch Dataset               Save Best Model
  ↓
DataLoader (batched)

MODEL ARCHITECTURE
──────────────────

Input: Cart sequence (max 6 items) + Candidate item
  ↓
Embedding Layer (128-dim)
  ↓
Positional Encoding
  ↓
Transformer Encoder (2 layers)
  - Multi-head attention (4 heads)
  - Feed-forward network
  - LayerNorm + Residual
  ↓
Mean Pooling
  ↓
Feature Fusion (cart + candidate + context)
  ↓
MLP Scoring Head (256 → 128 → 64 → 1)
  ↓
Sigmoid Output [0, 1]

  - Better than Popularity (7.68%)

NDCG@8 (Normalized Discounted Cumulative Gain)
  - Ranking quality considering position importance
  - Range: [0, 1] where 1 = perfect ranking
  - Transformer: 0.156 (achieves 15.6% of ideal ranking)
  - Industry standard for ranking evaluation

AUC (Area Under ROC Curve)
  - Binary classification quality
  - Range: [0.5, 1.0] where 0.5 = random, 1.0 = perfect
  - Transformer: 0.758 (very good classification)

Solution:

  $ pip install torch torchvision torchaudio  # Optional: GPU version

Problem: "CUDA out of memory"
Solution:
  1. Reduce batch_size in main.py:
     batch_size = 32  # Instead of 64
  2. Close unnecessary applications
  3. Use CPU instead:
     In train.py: device = 'cpu'

Problem: "FileNotFoundError: training_data.csv"
Solution:
  $ python data_generation.py  # ~7 minutes

Problem: Slow training on CPU
Solution:
  1. Install CUDA and PyTorch GPU version
  2. Or reduce epochs: num_epochs = 5
  3. Or skip ablation study (Optional)

Problem: "Out of memory during preprocessing"
Solution:
  1. Close other programs
  2. Process in smaller chunks:
     In data.py, add:
     df = df.sample(frac=0.5)  # Use 50% of data

EXAMPLE: GETTING PREDICTIONS FOR A SINGLE CART
═══════════════════════════════════════════════════════════════════════════════

from data import load_raw_datasets, CartRecommendationDataset
from model import TransformerRecommender
import torch

# Setup
device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
model = TransformerRecommender(config).to(device)
model.load_state_dict(torch.load("csao_model_v1.pth"))
model.eval()

# Sample order data
user_id = 42
restaurant_id = 17
cart_items = [105, 203, 301]           # Items already in cart
candidate_items = list(range(100, 150)) # Potential add-ons
timestamp = datetime.now()

# Preprocess
cart_padded = [105, 203, 301] + [-1] * 3  # Pad to length 6
user_features = [0.5, 0.3, 0.8, 0.2, 1.5]  # Normalized
rest_features = [0.0, 1.0]                 # One-hot cuisine
context_features = [12, 0]                 # Hour=12, not weekend

# Get predictions for each candidate
scores = []
for candidate in candidate_items:
    cart_t = torch.LongTensor([cart_padded]).to(device)
    cand_t = torch.LongTensor([[candidate]]).to(device)
    user_t = torch.FloatTensor([user_features]).to(device)
    rest_t = torch.FloatTensor([rest_features]).to(device)
    ctx_t = torch.FloatTensor([context_features]).to(device)
    
    with torch.no_grad():
        score = model(cart_t, cand_t, user_t, rest_t, ctx_t)
    scores.append(score.item())

# Get top 8
top_8_indices = sorted(range(len(scores)), key=lambda i: scores[i], reverse=True)[:8]
top_8_items = [candidate_items[i] for i in top_8_indices]
top_8_scores = [scores[i] for i in top_8_indices]

print("Top 8 Recommended Add-Ons:")
for rank, (item, score) in enumerate(zip(top_8_items, top_8_scores), 1):
    print(f"{rank}. Item {item}: {score:.4f} confidence")
