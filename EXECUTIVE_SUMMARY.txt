  - Mean pooling for variable-length carts
  - Multi-feature fusion network

✓ Proper Ranking Evaluation
  - Per-cart evaluation (realistic ranking metrics)
  - NDCG@K (normalized discounted cumulative gain)
  - Precision@K and Recall@K
  - AUC for binary classification
  - Per-cart metrics aggregation

  - Strict temporal split (no future leakage)
  - Feature normalization using train statistics
  - Cart sequence padding with attention masking
  - 1.05M features merged across users/restaurants/items

  - 3500+ lines of well-documented Python
